## Per-topic file pattern

As topics get filled in, each major one trends toward:

```
readme.md              # What it is and why it matters
Mental-Model.md        # Beginner-friendly explanation
Cheatsheet.md          # Commands / key ideas
Architecture.md        # How it works in production
Security.md            # Risks and mitigations
Cost.md                # Cost considerations
Troubleshooting.md     # Common failures and fixes
Interview-Questions.md # Role prep
```
