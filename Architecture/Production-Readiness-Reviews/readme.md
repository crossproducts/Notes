# Production Readiness Reviews

> **Status:** 🔴 Pending — skeleton created, content to be filled in.

## Purpose

A Production Readiness Review (PRR) is a structured gate before a service goes live —
it forces the operability, reliability, and security questions early.

## Template

```markdown
# PRR: <service>

## Overview
## Architecture diagram
## Reliability — SLOs, failure modes, DR
## Observability — metrics, logs, traces, alerts, dashboards
## Security — authn/authz, secrets, scanning
## Operations — runbooks, on-call, rollback
## Capacity — load tested, limits set
## Cost — estimate, budget alerts
## Action items
```
