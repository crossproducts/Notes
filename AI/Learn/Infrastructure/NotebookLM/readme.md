# NotebookLM

> [!NOTE]
> **Status**: Pending

---

> AI-powered research and note-taking tool by Google that grounds responses in your uploaded documents.

## Overview

| Property | Detail |
|---|---|
| Type | AI Research Assistant |
| Interface | Web UI |
| Platforms | Web (browser-based) |
| Website | https://notebooklm.google.com |
| Pricing | Free (Google account required) |

## Key Features

- Upload documents (PDFs, Google Docs, web URLs, text files)
- AI answers grounded exclusively in your sources
- Auto-generates summaries, study guides, and FAQs
- Audio Overview: generates podcast-style audio summaries
- Interactive notebook with cited responses

## Use Cases

- Research synthesis across multiple documents
- Study aid for dense technical material
- Meeting notes summarization
- Literature review assistance

## Getting Started

1. Go to https://notebooklm.google.com
2. Sign in with a Google account
3. Create a new notebook
4. Upload sources (PDFs, Docs, URLs, text)
5. Ask questions or generate summaries
